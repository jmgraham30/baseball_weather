# R Scripts for Project on Open-Air Baseball Parks

- [`tidyverse`](https://www.tidyverse.org/) 

- [`tidymodels`](https://www.tidymodels.org/) 

- [`janitor`](https://sfirke.github.io/janitor/)  

- [`kableExtra`](https://haozhu233.github.io/kableExtra/) 

- [`patchwork`](https://patchwork.data-imaginist.com/) 

- [`parttree`](https://github.com/grantmcdermott/parttree)

- [`RColorBrewer`](https://r-graph-gallery.com/38-rcolorbrewers-palettes.html) 
- [`vip`](https://koalaverse.github.io/vip/articles/vip.html) 

- [`rpart`](https://cran.r-project.org/web/packages/rpart/rpart.pdf) 

- [`rpart.plot`](http://www.milbo.org/rpart-plot/prp.pdf)  

- [`retrosheet`](https://www.retrosheet.org/)

- [`furrr`](https://furrr.futureverse.org/)

- [`GGally`](https://ggobi.github.io/ggally/)

- [`skmir`](https://cran.r-project.org/web/packages/skimr/vignettes/skimr.html)
